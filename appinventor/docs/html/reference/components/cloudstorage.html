Dropbox

Component for sync data on App Inventor to Dropbox.

Properties

AppKey
AppKey property setter method: sets the AppKey to be used when authorizing with Dropbox via OAuth.
AppSecret
AppSecret property setter method: sets the App Secret to be used when authorizing with Dropbox via OAuth.
DropboxUploadFolder
Return the Dropbox folder(directory) to where the uploadedfile will be placed
ScheduleUploadEnabled
Indicates whether there exists any schedule upload task
UploadPeriod
Indicates the interval for a re-occurring upload activity for uploading file to dropbox
WifiOnly
Indicates whether the user has specified that the DropBox could only use Wifi
appSecret
AppSecret property getter method.

Events

IsAuthorized()
Indicates when the authorization has been successful.
ServiceStatusChanged(successful,log)
This event is raised when upload service status has changed
UploadDone(successful)
Indicates when the authorization has been successful.

Methods

Authorize()
Start WebAuthentication session by logging in to Dropbox with a username and password.
CheckAuthorized()
Check whether we already have a Dropbox access token
DeAuthorize()
Remove authentication for this app instance
GetScheduleTaskLog()
Get the message log of the most recent schedule upload task
GetScheduleTaskLogTime()
Get the finshed datetime of last upload service task
GetScheduleTaskStatus()
Get the status of the status of the most recent schedule upload task
ScheduleUpload(folderPath,period)
Enable upload scheduled task based on specified filepath of a folder locally in parameter folderPath. One use case is to upload all the save photosin some SD folder periodically. The parameter period is in second.
ScheduleUploadDB(dbName,period)
Enable upload scheduled task based on specified filepath of a folder locally. One use case is to upload all the save photosin some SD folder periodically
StopScheduleUpload()
Stop the schedule uploading task
UploadDB(dbName)
Enable to upload the specified db file to remote storage place for backup. Will first archive the database. By default, if itâs database name forsensor compoent, it should be âSensorDataâ
UploadData(filepath)
This function uploads the file (as specified with its filepath) to dropbox folder.

GoogleDrive

Component for GoogleDrive

Properties

GoogleDriveFolder
Return the Google Drive folder(directory) to where the uploadedfile(s) will be placed
WifiOnly
Indicates whether the user has specified that the GoogleDrive component could only use Wifi to upload data

Events

IsAuthorized()
Indicates when the authorization has been successful.
ServiceStatusChanged(target,successful,log,time)
This event is raised when background upload tasks status for a has changed. If successful, successful will return true, else it returns false and log returns the error messages
UploadDone(successful)
Indicates when the upload task has been successful done.

Add a background task for uploading file(s) to Google Drive. The task will upload the file(s) that is specified in target to the Google drive folder specified in driveFolder with every period interval. Save taskName for later reference for removing the task
